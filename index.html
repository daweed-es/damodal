Damodal i a jQuery plugin that opens modal/dialog windows. Good perfomance with responsive desings.

USAGE

Link jquery.damodal.js and jQuery in the header section, asign the element to open the dialog and specify which dialog will open.

<html>
<head>
  <link rel="stylesheet" href="jquery.damodal.css" type="text/css"/>
  <script src="jquery-1.9.1.js"></script>
  <script src="jquery.damodal.js"></script>
  <script>
    $(document).ready(function(){
        $("#trigger1").damodal({modalWidth:"50%"});
    });
  </script>  
</head>
<body>
  <a href="#" id="trigger1" data-damodal="damodal1">Open modal</a>
  <div id="damodal1" class="modal-content">
      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  </div>
</body>
</html>
